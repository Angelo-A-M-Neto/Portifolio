#Meu Portfólio

Bem-vindo ao repositório do meu portfólio! Este projeto foi desenvolvido para apresentar meus conhecimentos, projetos e habilidades na área de desenvolvimento de software e tecnologia.

##Sobre

Este portfólio reúne alguns dos meus principais projetos, experiências e informações profissionais de forma organizada e responsiva.

O objetivo é demonstrar minha evolução como desenvolvedor, além de servir como um ponto central para conhecer meu trabalho.

##Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Git
- GitHub
- Mais

##Funcionalidades

- Página inicial com apresentação
- Seção sobre mim
- Exibição de projetos
- Lista de habilidades
- Contato



##Objetivos do Projeto

- Consolidar conhecimentos em desenvolvimento web.
- Criar uma apresentação profissional dos meus projetos.
- Demonstrar organização, boas práticas e versionamento com Git.
